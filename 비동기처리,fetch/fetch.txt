

비동기 처리 : 특정 코드의 연산이 끝날 때까지 코드의 실행을 멈추지 않고 다음 코드를 먼저 실행하는 자바스크립트의 특성
	    =====>>>> 특정 코드를 기다리지 않고 다음 코드를 실행

여기서 그럼 왜 다 끝나지도 않았는데 다음 코드를 실행 하냐?????

왜냐하면 서버가 대량의 데이터를 보낸다하면 마냥 기다릴수가 없다 너무~~~ 오래 걸린다.
그래서 다음 코드를 실행하는 것이다.


====== 비동기 처리의 문제 보안법  ======

1.콜백 함수
 
콜백 함수란 ??????
다른 함수의 인자거나 특정 이벤트에 의해서 실행되는 함수
콜백 함수는 특정 코드의 작업이 끝나야 실행되는 함수이다

2.promise() : 자바스크립트 비동기 처리에 사용되는 객체 ?????  즉 비동기 처리의 문제의 보안법 중 하나


3.async, await : 비동기처리의 문제를 해결하기 위한 보안법 중 최근 문법


=============================================================
fetch : JavaScript에서 서버로 네트워크 request를 보내고 response를 받을수 있게 해주는 매서드
        기본값 : GET
fetch().then() : fetch로 서보로 request를 보내고 response를 받으면 then()을 실행해줘.


=====>>>> fetch로 api와 연결하고 백엔드로부터 정보를 받으면 받은 정보로 then()을 실행해라

.json() : JavaScript Object Notation으로 데이터를 저장하거나 전송하는 Data저장방식

보통 fetch에서 한번, json형태로 얻을 때 한번 씩 await를 사용
