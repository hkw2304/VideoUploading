런타임환경 : 프로그래밍 언어가 실행되는 환경
            javascript의 런타임환경은 브라우저,node.js등등

node.js : 브라우저에서만 실행되는 javascript를 서버에서 실행 될 수 있게하는 런타임
express : node.js에서 개발을 편하게 할 수 있도록 도와주는 클래스와라이브러리 집합체인 프레임워크
pug : express view engine으로 html문법을 pug문법으로 작성하여 정적인페이지(브라우저)에 출력한다.
view engine : 서버에서 처리한 데이터결과를 브라우저에게 편히하게 출력하기위해 사용되는 것
webpack : 웹 사이트에 접속시 굉장히 많은 데이터들이 다운받아지는데 그러면 쓸데없는 자원도 낭비되고 시간도
          낭비되서 이러한 데이터들을 하나로 묶는다는 의미로 번들러로서 등장하게 됐다.
           이중에서 html, css, javascript등을 하나로 묶어주는 webpack있다.
